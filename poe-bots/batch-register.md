# Batch Registration Guide -- 40 Poe Bots

Quick-reference for manually creating all 40 bots on poe.com.
Each bot is a **Server bot** pointing to your deployed server.

---

## Before You Start

1. Deploy your server (Modal/VPS/Railway) -- see `deploy-guide.md`
2. Note your **Server URL**: `___________________________`
3. Note your **Access Key**: `___________________________` (from poe.com/api_key)
4. Open poe.com/create_bot in your browser
5. Keep this file open as reference

**Estimated time:** ~3 minutes per bot = ~2 hours total (or ~30 min per batch)

---

## Registration Flow (repeat for each bot)

1. Go to **poe.com/create_bot**
2. Select **"Server bot"**
3. Fill in: Handle, Display Name, Description, Server URL, Access Key
4. Add Server Bot Dependency (model)
5. Enable Markdown rendering
6. Click **Create**
7. Move to next bot

## After Registration

1. **Test each bot:** Click on the bot name, try the suggested replies
2. **Set visibility:** Bot Settings > Visibility > Public
3. **Enable monetization:** poe.com/settings > Creator Monetization
4. **Monitor:** Check server logs for errors with `modal app logs hanabi-poe-bots`
